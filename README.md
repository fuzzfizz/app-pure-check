# PureCheck 🌿

An AI-powered cosmetic and skincare ingredient safety analyzer tailored for Thai consumers. **PureCheck** allows users to scan cosmetic barcodes, extract ingredient lists, and get personalized safety analysis matching their unique skin type, conditions, and allergy history using Gemini AI.

---

## ✨ Features

* **Journey A: Personalized Skin Profile**
  * 4-step onboarding stepper to configure:
    * **Skin Type:** Oily, Dry, Combination, Normal, Sensitive.
    * **Skin Conditions:** Eczema, Rosacea, Psoriasis, Acne-prone, or Pregnancy/Breastfeeding health flags.
    * **Allergen Registry:** Register known allergens with severity levels and reaction symptoms.
    * **Concerns & Avoidances:** Select concerns (Acne, Wrinkles, Pores) and preferences (e.g., avoid fragrance/alcohol).
* **Journey B: Smart Barcode Camera Scanner**
  * Live camera barcode scanner powered by `mobile_scanner`.
  * Open Beauty Facts API fallback for unrecognized products.
  * Manual entry form with comma-separated parser for products not found online.
  * Verify & edit details page to inspect parsed ingredients list before analyzing.
* **Journey C: AI Safety Report (Gemini AI)**
  * **Traffic-Light Verdict:** Green (Safe), Yellow (Caution), Red (Allergen Found).
  * **Allergy Alarm:** Displays prominent warning cards if a registered allergen is detected in the product.
  * **AI Analysis Summary:** Returns natural language explanations in Thai generated by `gemini-1.5-flash`.
  * **Interactive Breakdown:** Grouped list of ingredients (Safe / Caution / Danger) with toggleable functionality descriptions.
  * **Quick Allergen Report:** Instantly flag and add a suspicious ingredient to your personal allergen registry.
* **Journey D: Discovery & Search**
  * Unified search bar to query local database products and individual chemical ingredients.
  * Detail pages for products and ingredients with one-click profile checks.
* **Journey E: Profile & Settings**
  * Update skin profile, manage allergen lists, view historical scan logs, and securely log out.

## 🛠️ Tech Stack & Architecture

* **Framework:** Flutter 3.44.1 / Dart 3.12.1
* **State Management:** Riverpod (`flutter_riverpod`)
* **Navigation:** `go_router` (enforcing auth redirect guards)
* **Backend Database & Auth:** Supabase (`supabase_flutter`)
* **AI Engine:** Google Generative AI (`google_generative_ai` / Gemini 1.5 Flash)
* **Design System:** Custom Material 3 theme using **Noto Sans Thai** (Google Fonts)
* **Localizations:** English and Thai bilingual support (ARB compiler via `flutter gen-l10n`)

## 🚀 Getting Started

### 1. Prerequisites
Ensure you have the Flutter SDK configured on your machine.
```bash
flutter doctor
```

### 2. Configure Environment Variables
1. Copy the template file:
   ```bash
   cp .env.example .env
   ```
2. Open `.env` and fill in your Supabase and Gemini credentials:
   ```env
   SUPABASE_URL=https://your-project.supabase.co
   SUPABASE_ANON_KEY=your_anon_key_here
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

### 3. Install Dependencies & Generate Code
Install package dependencies:
```bash
flutter pub get
```
Compile localization files:
```bash
flutter gen-l10n
```

### 4. Run the Application
Start the development server on a connected mobile device or emulator:
```bash
flutter run
```
